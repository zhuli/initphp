Helper类使用说明：

  • 如果你开发的项目中，设计的类和业务有关系不能放进框架的Library下面，而且具备一定的通用性不能放进Service中，则可以放置在    library/helper/目录下面
  • Helper类：项目中公用的类，但是不适合放入InitPHP的Library扩展类库中，也不适合用Service

如何使用Helper类：

  • InitPHP框架提供了InitPHP::import()和InitPHP::loadclass()两个函数，来导入类文件和实例化类
public function run() {
	InitPHP::import('library/helper/test.php');
	$a = InitPHP::loadclass('test');
	$a->test();
	$this->view->set_tpl('test'); //设置一个test.htm的模板页面
	$this->view->display(); //模板显示
}
