工具描述：

先进先出的队列小工具。属于InitPHP框架工具类，需要通过$this->getUtil()方法获取

使用方法：

$queue = $this->getUtil('queue');

接口描述：

接口：$queue->set($val) 设置值
参数 类型 是否必须 描述
$val String 是 设置值
接口：$queue->get() 设置值
参数 类型 是否必须 描述
接口：$queue->count() 统计有多少个数
参数 类型 是否必须 描述
接口：$queue->clear() 清除数据
参数 类型 是否必须 描述

实例：

class indexController extends Controller {
	
	public $initphp_list = array('test'); 
	
	public function run() {
		$queue = $this->getUtil('queue'); 
		$queue->set('v1');
		$queue->set('v2');
		$queue->set('v3');
		$queue->set('v4');
		echo $queue->get();
		$this->view->display(); //模板显示
	}
	
	public function test() {
		echo 'Hello World';
	}
	
	/**
	 * @return testService
	 */
	private function getTestService() {
		return InitPHP::getService('test','test');
	}

}
