工具描述：

在开发的时候可以用来查看代码中具体运行了多少条SQL语句。有助于帮助开发者在复杂页面检测SQL情况。属于InitPHP框架工具类，需要通过$this->getUtil()方法获取

使用方法：

$sqlcontrol = $this->getUtil('sqlcontrol');

接口描述：

接口：$sqlcontrol->start() 数据库语句监控器-开始点
参数 类型 是否必须 描述
接口：$sqlcontrol->end() 数据库语句监控器-结束点
参数 类型 是否必须 描述

实例：

class indexController extends Controller {
	
	public $initphp_list = array('test'); 
	
	public function run() {
		$sqlcontrol = $this->getUtil('sqlcontrol'); 
		$sqlcontrol->start();
		$this->getTestService()->test();
		$sqlcontrol->end();
		$this->view->display(); //模板显示
	}
	
	public function test() {
		echo 'Hello World';
	}
	
	/**
	 * @return testService
	 */
	private function getTestService() {
		return InitPHP::getService('test','test');
	}

}
